Translate files .po

This module allows you to translate applications and models from the translation export wizard.

Language export wizard

New fields have been added in the language export.

Result

the translated file can be downloaded in the wizard.

View and/or modify file

By opening the file in this case in POEDIT we can view and edit the translated terms.

Note

  • It is necessary to install the following libraries: deep-translator - polib
  • If you are translating very large applications it is possible that the translation takes too long and therefore stops the Odoo service. If that happens consider extending the timeout in your configuration file.
